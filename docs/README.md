## 🛠 Todo アプリの特徴

このプロジェクトは学習目的で作成された Rails アプリケーションで、以下の技術とベストプラクティスを実装しています：

### 技術スタック

- **Rails 7.2** - 最新の Rails フレームワーク
- **MySQL 8.0** - リレーショナルデータベース
- **Docker** - コンテナ化技術
- **Tailwind CSS** - モダン CSS フレームワーク
- **Turbo & Stimulus** - Rails の JavaScript ソリューション

### 実装機能

- ✅ CRUD 操作（作成・読み取り・更新・削除）
- ✅ RESTful API 設計
- ✅ レスポンシブ UI
- ✅ データバリデーション
- ✅ 状態管理（完了/未完了）
- ✅ Docker での環境分離

### 学習ポイント

- **MVC アーキテクチャ**の実践
- **Active Record ORM** の活用
- **ERB テンプレート**エンジン
- **Rails 規約**の体験
- **Docker + Rails** 開発環境
